# 📊 Sales Analytics Project

An end-to-end **Sales Analytics Project** built using **SQL, Microsoft Excel, and Power BI** to transform raw retail sales data into meaningful business insights through data cleaning, exploratory data analysis (EDA), interactive dashboards, and KPI reporting.

---

## 📌 Project Overview

This project demonstrates the complete data analytics workflow, from raw sales data to interactive dashboards. It focuses on analyzing sales performance, product trends, regional performance, customer behavior, and the impact of discounts on profitability.

The project includes SQL-based data cleaning and analysis, an interactive Excel dashboard, and a Power BI dashboard to support data-driven business decisions.

## 📊 Key Business Insights

* Technology generated the highest sales among all product categories.
* The West region recorded the highest overall sales.
* A small number of products contributed significantly to total revenue.
* Moderate discounts resulted in better profitability than higher discounts.
* High discounts negatively impacted overall profit margins.
